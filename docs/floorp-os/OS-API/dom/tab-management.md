# タブ管理

## タブ管理

### タブ一覧の取得

タブの一覧を取得する。

**エンドポイント**: `GET /tabs/list`

**リクエスト**: パラメータなし。

**成功レスポンス** (200):

```json
[
  {
    "browserId": "string",
    "url": "string",
    "title": "string",
    "isActive": true,
    "isPinned": false,
    "isLoading": false
  }
]
```

**失敗レスポンス** (500):

```json
{
  "error": "string"
}
```

**内部動作**:

1. `gBrowser.tabs` から全タブを列挙する。
2. 各タブの `linkedBrowser` からプロパティを取得する。
3. 配列として返却する。

**備考**: 返却されるのはブラウザウィンドウ内の全タブ。インスタンスに紐付いていないタブも含まれる。

### インスタンスの作成

新しいタブを開き、インスタンスを作成する。

**エンドポイント**: `POST /tabs/instances`

**リクエスト**:

```json
{
  "url": "https://example.com",
  "inBackground": true,
  "waitForLoad": true
}
```

| パラメータ     | 必須     | 型        | デフォルト | 説明                                                                      |
| -------------- | -------- | --------- | ---------- | ------------------------------------------------------------------------- |
| `url`          | **はい** | `string`  | —          | 開く URL。`http`、`https`、`about:blank` のみ許可。                       |
| `inBackground` | いいえ   | `boolean` | `true`     | `false` の場合、タブをアクティブにする。                                  |
| `waitForLoad`  | いいえ   | `boolean` | `true`     | `false` の場合、ページの読み込み完了を待たずに即座に返却する（約 31ms）。 |

**成功レスポンス** (200):

```json
{
  "instanceId": "string"
}
```

**失敗レスポンス** (400):

```json
{
  "error": "url required"
}
```

**内部動作**:

1. URL のバリデーション（`http`/`https`/`about:blank`）。
2. `gBrowser.addTab()` でタブを作成する。
3. `waitForLoad=true` の場合、`STATE_STOP` イベントを待機する。
4. 一意の `instanceId` を生成してタブと紐付ける。
5. `instanceId` を返却する。

**備考**: `waitForLoad=false` を指定すると、ページの読み込みが完了していない状態でインスタンスが返される。以降の操作で要素が見つからない可能性がある。

### 既存タブへのアタッチ

既に開いているタブにインスタンスを紐付ける。

**エンドポイント**: `POST /tabs/attach`

**リクエスト**:

```json
{
  "browserId": "12345"
}
```

| パラメータ  | 必須     | 型       | デフォルト | 説明                                              |
| ----------- | -------- | -------- | ---------- | ------------------------------------------------- |
| `browserId` | **はい** | `string` | —          | タブの `browserId`。`GET /tabs/list` で取得可能。 |

**成功レスポンス** (200):

```json
{
  "instanceId": "string | null"
}
```

**失敗レスポンス** (400):

```json
{
  "error": "browserId required"
}
```

**内部動作**:

1. `browserId` に一致するタブを検索する。
2. 見つかった場合、新しいインスタンスを作成して紐付ける。
3. 見つからない場合、`instanceId: null` を返却する。

**備考**: 同一タブに複数のインスタンスをアタッチ可能。各インスタンスは独立して管理される。

### インスタンス情報の取得

インスタンスの詳細情報を取得する。

**エンドポイント**: `GET /tabs/instances/:id`

**パスパラメータ**:

| パラメータ | 必須     | 型       | 説明              |
| ---------- | -------- | -------- | ----------------- |
| `id`       | **はい** | `string` | インスタンス ID。 |

**リクエスト**: パラメータなし。

**成功レスポンス** (200):

```json
{
  "instanceId": "string",
  "url": "string",
  "title": "string",
  "favIconUrl": "string",
  "isActive": true,
  "isPinned": false,
  "isLoading": false,
  "html": "string",
  "screenshot": "string"
}
```

| フィールド   | 型        | 説明                           |
| ------------ | --------- | ------------------------------ |
| `instanceId` | `string`  | インスタンス ID。              |
| `url`        | `string`  | 現在の URL。                   |
| `title`      | `string`  | ページタイトル。               |
| `favIconUrl` | `string`  | ファビコンの URL。             |
| `isActive`   | `boolean` | アクティブタブかどうか。       |
| `isPinned`   | `boolean` | ピン留めされているかどうか。   |
| `isLoading`  | `boolean` | 読み込み中かどうか。           |
| `html`       | `string`  | ページの HTML ソース。         |
| `screenshot` | `string`  | スクリーンショット（Base64）。 |

**失敗レスポンス** (404):

```json
{
  "error": "Instance not found"
}
```

**備考**: `html` と `screenshot` の取得はコストが高い。頻繁なポーリングには `GET /tabs/instances/:id/exists` を推奨する。

### インスタンスの破棄

インスタンスを破棄する。ブラウザタブは閉じない。

**エンドポイント**: `DELETE /tabs/instances/:id`

**パスパラメータ**:

| パラメータ | 必須     | 型       | 説明              |
| ---------- | -------- | -------- | ----------------- |
| `id`       | **はい** | `string` | インスタンス ID。 |

**リクエスト**: パラメータなし。

**成功レスポンス** (200):

```json
{
  "ok": true
}
```

**失敗レスポンス** (404):

```json
{
  "error": "Instance not found"
}
```

**内部動作**:

1. インスタンス管理マップからエントリを削除する。
2. 関連リソース（オブザーバ等）をクリーンアップする。
3. タブ自体は開いたまま残る。

**備考**: 破棄後もタブはブラウザに残る。タブも閉じたい場合は `POST /tabs/instances/:id/close` を使用する。

---

インスタンスを破棄し、ブラウザタブも閉じる。

**エンドポイント**: `POST /tabs/instances/:id/close`

**パスパラメータ**:

| パラメータ | 必須     | 型       | 説明              |
| ---------- | -------- | -------- | ----------------- |
| `id`       | **はい** | `string` | インスタンス ID。 |

**リクエスト**: パラメータなし。

**成功レスポンス** (200):

```json
{
  "ok": true
}
```

**失敗レスポンス** (404):

```json
{
  "error": "Instance not found"
}
```

**内部動作**:

1. インスタンス管理マップからエントリを削除する。
2. 関連リソースをクリーンアップする。
3. `gBrowser.removeTab()` でブラウザタブを閉じる。

### インスタンスの存在確認

インスタンスが存在するかを軽量に確認する。

**エンドポイント**: `GET /tabs/instances/:id/exists`

**パスパラメータ**:

| パラメータ | 必須     | 型       | 説明              |
| ---------- | -------- | -------- | ----------------- |
| `id`       | **はい** | `string` | インスタンス ID。 |

**リクエスト**: パラメータなし。

**成功レスポンス** (200):

```json
{
  "exists": true
}
```

**内部動作**:

1. インスタンス管理マップにキーが存在するかを確認する。

**備考**: 応答時間は 3ms 未満。ポーリングに適している。
